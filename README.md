# Site Starter
> Start a frontend site with Webpack, Babel, ES6 or TypeScript, critical CSS & differential loading

### Installation

```
npm install
```

### Start DevServer with Hot Module Replacement

```
npm start
```

### Build

```
npm run build

npx http-server ./build
```

### Features:

* Babel & webpack
* ES6 & TypeScript
* CSS & SASS
* ESLint
* Critical
* Differential loading
